# responsive-learning-journal
A mobile first approach journal showcasing responsive design

## About the Codebase

This project is a responsive learning journal website that showcases frontend development projects. It follows a mobile-first design approach and demonstrates various JavaScript concepts and responsive web design techniques.

### Project Structure

- **`index.html`**: The main HTML file containing the structure of the learning journal, including header, navigation, main content sections, and footer.
- **`index.css`**: Stylesheet providing responsive design and styling for the journal, using the Source Sans 3 font family and Font Awesome icons.
- **`index.js`**: JavaScript file that handles dynamic content rendering, date display, and navigation functionality. It imports data from `data.js` and renders journal entries.
- **`data.js`**: Contains an exported array of journal entries, each with project details including title, description, image, and GitHub link.
- **`images/`**: Directory containing project screenshots and the about section image.
- **`wireframe/`**: Directory likely containing design mockups or wireframes for the project.

### Technologies Used

- **HTML5**: For semantic markup and structure
- **CSS3**: For responsive styling and layout
- **JavaScript (ES6+)**: For dynamic content and interactivity
- **Font Awesome**: For icons
- **Google Fonts**: For typography (Source Sans 3)

### Features

- Mobile-first responsive design
- Dynamic journal entry rendering from data
- Navigation between home and about sections
- Links to external GitHub project repositories
- Real-time date display

### How to Run

1. Clone the repository
2. Open `index.html` in a modern web browser
3. The journal will load with the current date and project entries

No build process or server is required as this is a static website.
